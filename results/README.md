# MultiRel — Experimental Results

Raw per-seed results for all experiments in the paper.
All values are MSE on the test split under the standard Time-Series-Library protocol
(`seq_len=96`, `pred_len` varies, 3 independent random seeds averaged).

## Files

| File | Contents |
|------|----------|
| `grid.json` | Multi-horizon MSE (pred-24/48/96) for PEMS03/04/07/08 — iTransformer vs MultiRel |
| `pred12.json` | pred-12 baselines: DLinear, PatchTST, iTransformer, MultiRel on all 4 PEMS datasets |
| `ablation.json` | Single-relation ablations (partial-corr only / Pearson only / lead-lag only) on PEMS04/08 pred-12 |
| `stability.json` | Training stability at lr=5×10⁻⁴ on PEMS07/03, demonstrating MultiRel's regularisation benefit |
| `pred96_extra.json` | pred-96 on PEMS04/08 (supplemental; PEMS08 shows high instability for both models) |
| `ksens.json` | k-sensitivity for PEMS04 pred-12 (k=5 verified; k=10/15/25/30 pending) |
| `graphstats.json` | Precomputed graph densities and community counts (k=20) across all 4 datasets |
| `raw_logs/` | Verbatim training logs from GPU boxes (source of truth for all numbers above) |

## JSON schema

Each file has this structure:
```json
{
  "experiment": "<name>",
  "description": "<one-line>",
  "protocol": { ... },
  "results": [
    {
      "dataset": "PEMS04",
      "N": 307,
      "model": "MultiRel",
      "pred_len": 12,
      "seeds_mse": [0.07930, 0.07928, 0.07964],
      "seeds_mae": [...],
      "mean_mse": 0.0794
    }
  ]
}
```

Per-seed values allow replication of the 3-seed mean reported in the paper.
Where seeds are diverged (MSE≫expected), a `"note"` field explains.

## Reproducing

See the paper's Appendix / Section 4 for the full training command.
The model code and precompute scripts are in the companion code repository.
Graph masks (`.npy`) are generated by `precompute_graphs.py` using only the training split.
